# Wake Word Models

Wake word models are bundled with the app.

## Files

- `okay_nabu.json` / `okay_nabu.tflite` - Wake word "Okay Nabu"
- `hey_mycroft.json` / `hey_mycroft.tflite` - Wake word "Hey Mycroft"
- `hey_jarvis.json` / `hey_jarvis.tflite` - Wake word "Hey Jarvis"
- `stop.json` / `stop.tflite` - Stop word for canceling

These MicroWakeWord models are aligned with
[OHF-Voice/linux-voice-assistant](https://github.com/OHF-Voice/linux-voice-assistant).
